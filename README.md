# Sample Scout API System

Sample Scout is a Python-based audio sample discovery project designed to scan sample libraries, extract metadata, and make sample collections searchable.

## Current Features

* Recursively scans audio sample folders
* Detects WAV, MP3, AIFF, and FLAC files
* Extracts metadata including category, filename, extension, and file path
* Stores metadata in a CSV database
* Supports searching by category
* Supports searching by filename

## Technologies

* Python
* pathlib
* csv
* Git
* GitHub

## Example Workflow

1. Scan a sample library folder
2. Extract metadata from audio files
3. Generate a searchable CSV database
4. Search samples by category or filename
